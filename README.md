# 整体环境器具

## esphome安装

使用pip安装esphome,安装即可仅使用其dashboard能力，一般做一下yaml格式化校验

- esphome dash board [yaml文件目录]

## 用github Actions编译

因为esphome编译过程需要下载很多相应资源，但因为网络限制往往比较难下载。所以借助Github的Action进行云编译.

参考工程：https://github.com/ericzhang1124/my-esphome-config


## web.esphome.io

https://web.esphome.io/ web工具，用来烧录bin固件版本到开发板
